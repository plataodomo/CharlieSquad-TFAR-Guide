1) Requirements

  • Arma 3
  • TeamSpeak 3 (64-bit)
  • Arma 3 Launcher (comes with the game)

2) Download & Install TFAR (BETA!!!)

  1. Open Steam.
  2. Go to the Arma 3 Workshop.
  3. Search for Task Force Arrowhead Radio (BETA!!!).
  4. Click Subscribe. The mod will download to:
    ...\Steam\steamapps\workshop\content\107410\620019431

3) Enable the Mod in Arma 3

  1. Launch the Arma 3 Launcher.
  2. Open the Mods tab.
  3. Find @Task Force Arrowhead Radio (BETA!!!).
  4. Tick it to load with the game (consider saving a preset).

4) Install the TeamSpeak Plugin

  1. Open the TFAR mod folder (right-click in Launcher → Open Folder) or go to:
    ...\Steam\steamapps\workshop\content\107410\620019431
  2. Open: @Task Force Arrowhead Radio (BETA!!!)\teamspeak
  3. Double-click task_force_radio.ts3_plugin.
  4. Approve the install in TeamSpeak, then restart TeamSpeak.
  5. In TeamSpeak: Tools → Options → Addons → Plugins → ensure Task Force Arrowhead Radio is enabled.

5) First-Time Setup

  1. ⬇ Download TeamSpeak 3 (64-bit)
  2. Start TeamSpeak 3 first.
  3. Connect to our TS server: CharlieSquad.4npserver.com
  4. Start Arma 3 with TFAR enabled.
  5. Join our Arma server: 85.14.210.14:22500
  6. On success, TeamSpeak should display: “Task Force Radio connected to game”.

6) Troubleshooting

  • No TFAR popup in TS? Plugin not installed or not enabled.
  • Game ↔ TS not syncing? Ensure TeamSpeak is 64-bit (same as Arma).
  • Missing keybinds? In Arma: Controls → Configure Addons → TFAR to set radio keys.
  • Still having problems? Ask for help in our Discord server.
💡 Tip: Keep TeamSpeak open before launching Arma so TFAR can hook immediately.
